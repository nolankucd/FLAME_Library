FLAME Digital Library

Explore interactive 3D concepts from across STEAM subjects to enhance understanding.

The FLAME project is a SATLE funded project that aims to train educators with no experience in 3D modelling, animation and web developent with the skills needed to create their own interactive 3D web content. FLAME stands for "Facilitating Learning with Animation and Multimedia Engagement"
