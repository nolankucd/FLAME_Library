Tutorial: Scan an object with your smartphone

You can use various tools to scan an object with your smartphone. On iOS you can use Apple's Reality Composer app. This makes capturing a 3D model of an object very easy. Use Blender to open the USDZ file generated by Reality Composer.

Export it as a binary glb file and upload to GitHub and include in your GitHub page. See the source of this page to see how to implement the html.
